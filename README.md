# HostMigrationSample
## サンプルの説明
Assets/Scenes/Joinシーンを再生してJoinボタンを押せば，セッションがスタートします．
## パッケージの使い方
1. RunnerにObserverをアタッチする
2. 復元したいNetworkObjectにResumeObjectをアタッチする
